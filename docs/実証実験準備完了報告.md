## 🚀 実証実験の開始方法

### クイックスタート

1. **セットアップ**
   ```bash
   npm install
   cd frontend && npm install && cd ..
   ```

2. **環境変数の設定**
   - `frontend/.env.local` を作成
   - Airtable APIキーとBase IDを設定

3. **Airtableのセットアップ**
   ```bash
   npm run setup-airtable-auto
   ```

4. **サーバー起動**
   ```bash
   npm run dev
   ```

5. **データの投入**
   - Airtableに学生データを入力
   - または `/import-sample` からサンプルデータを投入

6. **実証実験の開始**
   - `docs/実証実験ガイド.md` を参照
   - `docs/実証実験前チェックリスト.md` で最終確認
