# 開発サーバーの起動方法

## 手動で起動する場合

1. ターミナル（PowerShell）を開く
2. 以下のコマンドを実行:

```powershell
cd C:\Users\USER\.cursor\pbl-ai-dashboard\frontend
npm run dev
```

3. ターミナルに表示されるURL（例: http://localhost:3000）をブラウザで開く

## ポートが使用されている場合

ポート3000が使用されている場合、自動的に3001、3002...と順番に試されます。
ターミナルに表示される正しいポート番号でアクセスしてください。

## トラブルシューティング

- 真っ白な画面が表示される場合: Ctrl+Shift+R でハードリロード
- エラーが表示される場合: ターミナルのエラーメッセージを確認
