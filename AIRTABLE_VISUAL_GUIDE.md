## 現在の状況

✅ 「Create your first base」ボタンをクリック済み
❓ 次の画面で「Start from scratch」が見つからない

## 次の画面で見るべきもの

### 画面1: テンプレート選択画面

以下のような画面が表示されている場合：

```
┌─────────────────────────────────────┐
│  Choose a template                  │
├─────────────────────────────────────┤
│  [Blank]  [Project tracker]         │
│  [Content calendar]  [CRM]          │
│  ...                                │
└─────────────────────────────────────┘
```

**→ 「Blank」をクリック！**

### 画面2: ベース名入力画面

以下のような画面が表示されている場合：

```
┌─────────────────────────────────────┐
│  Name your base                     │
├─────────────────────────────────────┤
│  [________________]                 │
│                                     │
│  [Create base]  [Cancel]           │
└─────────────────────────────────────┘
```

**→ ベース名を「PBL AI Dashboard」と入力して「Create base」をクリック！**

### 画面3: ワークスペース選択画面

以下のような画面が表示されている場合：

```
┌─────────────────────────────────────┐
│  Choose a workspace                 │
├─────────────────────────────────────┤
│  ○ My workspace                     │
│                                     │
│  [Continue]  [Cancel]               │
└─────────────────────────────────────┘
```

**→ ワークスペースを選択して「Continue」をクリック！**

## 🎯 最も確実な方法

### 方法1: 「Blank」を探す

1. 現在の画面で「**Blank**」という文字を探す
2. クリックする
3. ベース名を「**PBL AI Dashboard**」と入力
4. 「**Create base**」をクリック

### 方法2: 直接URLから

ブラウザのアドレスバーに以下を入力：

```
https://airtable.com/app
```

Enterキーを押すと、直接空のベースが作成される場合があります。

### 方法3: ダッシュボードに戻る

1. ブラウザの「戻る」ボタンで前の画面に戻る
2. または、Airtableのロゴをクリックしてダッシュボードに戻る
3. 画面の左上の「**+**」ボタンを探す
4. クリックして「**New base**」→「**Blank**」を選択

## 📸 画面の特徴

### 「Blank」や「Empty」の見つけ方

- **テンプレート一覧の一番上**にあることが多い
- **左側のタブ**として表示されることもある
- **「Start from scratch」**という名前で表示されることもある
- **「Empty base」**という名前で表示されることもある

### 見た目の特徴

- 通常は**グレー**または**白**の背景
- テンプレートと比べて**シンプル**なデザイン
- 「**+**」アイコンが表示されることが多い
