# Airtableベース作成：手動で確実に作成する方法

「Start from scratch」が見つからない場合、以下の手順で確実に作成できます。

## 🎯 方法1: ダッシュボードから作成（推奨）

### ステップ1: Airtableのダッシュボードに戻る

1. ブラウザの「**戻る**」ボタン（←）をクリック
2. または、Airtableの**ロゴ**（左上）をクリック
3. ダッシュボードが表示されるまで戻る

### ステップ2: 「+」ボタンを探す

ダッシュボードで以下を探してください：

#### 場所A: 画面の左上
```
┌─────────────────────────┐
│ [Airtable] [+ Add base] │  ← ここ
│                         │
│  Your bases            │
│  ...                    │
└─────────────────────────┘
```

#### 場所B: 画面の中央上部
```
┌─────────────────────────┐
│                         │
│  [+ Create a base]      │  ← ここ
│                         │
│  Your bases            │
│  ...                    │
└─────────────────────────┘
```

#### 場所C: 左サイドバーの一番下
```
┌─────────┬──────────────┐
│ Workspace│              │
│          │              │
│ Bases    │              │
│  ...     │              │
│          │              │
│ [+]      │              │  ← ここ
└─────────┴──────────────┘
```

### ステップ3: 「+」ボタンをクリック

1. 「+」ボタンをクリック
2. メニューが表示される：
   - 「**New base**」
   - 「**Add a base**」
   - 「**Create base**」
   など

### ステップ4: 「Blank」を選択

メニューから以下を選択：
- 「**Blank**」
- 「**Start from scratch**」
- 「**Empty base**」
- 「**New blank base**」

### ステップ5: ベース名を入力

1. ベース名に「**PBL AI Dashboard**」と入力
2. 「**Create base**」または「**Continue**」をクリック

---

## 🎯 方法2: 既存のベースから複製

既に他のベースがある場合：

### ステップ1: 既存のベースを開く

1. ダッシュボードで既存のベースを開く

### ステップ2: ベースを複製

1. ベース名の横にある「**...**」（三点リーダー）をクリック
2. 「**Duplicate base**」を選択
3. ベース名を「**PBL AI Dashboard**」に変更
4. 「**Duplicate**」をクリック

### ステップ3: 既存のテーブルを削除

1. 複製されたベースが開く
2. 既存のテーブルを削除：
   - テーブル名の横の「**...**」をクリック
   - 「**Delete table**」を選択
3. すべてのテーブルを削除

### ステップ4: 新しいテーブルを作成

1. 左下の「**+ Add a table**」をクリック
2. テーブル名を「**Students**」に変更
3. これで空のベースが完成

---

## 🎯 方法3: テンプレートから作成（最後の手段）

### ステップ1: テンプレートを選択

1. ダッシュボードで「**Browse templates**」をクリック
2. または、「**Templates**」タブを開く
3. 任意のテンプレートを選択（例：「Project tracker」）

### ステップ2: ベース名を変更

1. ベース名を「**PBL AI Dashboard**」に変更

### ステップ3: 既存のテーブルを削除

1. すべての既存テーブルを削除
2. 新しいテーブルを作成

---

## ✅ ベースが作成できたら

1. **Base IDをコピー**
   - ブラウザのアドレスバーを見る
   - URL: `https://airtable.com/appXXXXXXXXXXXXXX/...`
   - **`appXXXXXXXXXXXXXX`** の部分をコピー

2. **自動セットアップスクリプトを実行**
   ```powershell
   cd C:\Users\USER\.cursor\pbl-ai-dashboard
   npm run setup-airtable
   ```

スクリプトが以下を自動作成します：
- ✅ Studentsテーブル
- ✅ Tasksテーブル
- ✅ Teamsテーブル
- ✅ すべてのフィールド
