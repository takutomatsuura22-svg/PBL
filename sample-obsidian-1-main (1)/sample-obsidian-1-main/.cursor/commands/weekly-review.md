# 週次レビュー

## Description
週次レビューを実行します。

## Prompt
週次レビューを実行してください。

実行内容:
1. Inbox整理レポート生成
2. 孤立ノート検出
3. 今週作成したノート数をカウント
4. タグ使用状況の分析
5. 03_Inputのクリーンアップ（1週間触っていないファイル）
6. 来週のアクション提案
7. 週次レビューファイル作成: `02_Daily/Weekly-Reviews/YYYY/YYYY-W[週番号].md`

出力形式:
```markdown
# 週次レビュー (YYYY-MM-DD)

## 📊 今週の実績
- 作成ノート: X件
- Memory作成: Y件
- Daily継続: Z日
- Inputクリーンアップ: A件

## 📋 Inbox整理
[Inbox整理レポート]
- Memory候補: X件
- Input候補: Y件
- Output候補: Z件
- 要追加情報: A件
- 削除推奨: B件

## 🔗 孤立ノート
[孤立ノート一覧とリンク候補]

## 🏷️ タグ使用状況
Top 5タグ:
1. #technology (15回)
2. #meeting (8回)
3. #project/ProjectX (5回)
4. #memory (12回)
5. #technology/api (6回)

## 💡 来週のアクション
- [ ] Memory作成: 3件
- [ ] 孤立ノート解消: 全件
- [ ] Inbox整理: 未完了分を処理
- [ ] Inputクリーンアップ: 古いファイルを移動
```

手順:
1. 今週の日付範囲を計算（週番号も計算）
2. 作成されたノートをカウント
3. Inbox整理レポートを生成
4. 孤立ノートを検出
5. タグ使用状況を分析
6. 03_Input内の1週間触っていないファイルを検出
7. 週次レビューファイルを作成: `02_Daily/Weekly-Reviews/YYYY/YYYY-W[週番号].md`
8. 来週のアクションを提案
