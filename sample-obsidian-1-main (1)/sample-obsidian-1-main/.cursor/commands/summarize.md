# ノート要約

## Description
指定ノートを要約します。

## Prompt
指定ノートを要約してください。

引数:
- ファイル名（省略時は現在開いているファイル）

実行内容:
1. ノートの内容を読み込む
2. 3行で要約
3. キーポイントを3つ抽出
4. 関連ノートを提案

出力形式:
```markdown
## 要約
[3行要約]

## キーポイント
1. [ポイント1]
2. [ポイント2]
3. [ポイント3]

## 関連ノート
- [[関連1]]
- [[関連2]]
```

手順:
1. 指定ファイルを読み込む（ファイル名がない場合は現在開いているファイル）
2. 内容を分析して要約
3. 重要なポイントを3つ抽出
4. セマンティック検索で関連ノートを探す
5. 結果を表示
