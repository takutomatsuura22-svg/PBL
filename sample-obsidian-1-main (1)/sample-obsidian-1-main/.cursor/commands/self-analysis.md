# あきらパパ自己分析会議

## Description
あきらパパの自己分析を実施し、複数のキャラクターによる会議形式で深い洞察を得ます。

## Prompt
あきらパパの自己分析会議を実施してください。

## 実施内容

### 1. 情報収集
以下のフォルダから関連情報を収集：
- `01_Inbox/` - 最近の気づきやメモ
- `04_Memory/` - 特に以下のサブフォルダ：
  - `Personal/` - 個人関連の知識
  - `Business/` - ビジネス関連の知識
- `02_Daily/` - 最近の日次記録
- `05_Output/Projects/` - プロジェクト状況
- `05_Output/Areas/` - 継続的な活動状況

### 2. キャラクター設定
関連する4人のキャラクターを作成（例）：
- **エンジニア視点**: 技術者としてのスキル・キャリア
- **教育者視点**: 教育・指導の観点
- **ビジネス視点**: ビジネス戦略・収益化
- **家族・パーソナル視点**: プライベート・家族・健康

各キャラクターは異なる視点からあきらパパを分析

### 3. 会議実施
4人のキャラクターによる約5000文字の会議形式の分析を実施

**会議の構成:**
```markdown

- [キャラクター1]: [専門分野]
- [キャラクター2]: [専門分野]

- [キャラクター4]: [専門分野]

[会議の目的を明記]

### イントロダクション
[会議の開始と背景]

### セッション1: [トピック1]
[キャラクター1]: ...
[キャラクター2]: ...
[キャラクター3]: ...
[キャラクター4]: ...

[議論の詳細]

...

...

- [キャラクター1]: ...

## 関連ファイル
- [[04_Memory/Personal/...]]
- [[04_Memory/Business/...]]
- ...
```

### 4. 議事録保存
会議議事録を以下に保存：
- ファイル名: `04_Memory/Personal/Self-Analysis/YYYYMMDD_あきらパパ自己分析会議.md`
- 日付を自動挿入

## 手順

1. **情報収集**
   - `01_Inbox/`、`04_Memory/`、`02_Daily/`、`05_Output/Projects/`、`05_Output/Areas/`から最新情報を収集
   - 特にあきらパパに関する情報を抽出

2. **キャラクター作成**
   - 収集した情報をもとに4人の専門キャラクターを設定
   - 各キャラクターの視点を明確化

3. **会議実施**
   - 4人のキャラクターによる5000文字程度の会議形式の分析
   - 深い洞察と建設的な議論を展開

4. **議事録保存**
   - `04_Memory/Personal/Self-Analysis/YYYYMMDD_あきらパパ自己分析会議.md`に保存

5. **分析ファイル作成**
   - 議事録から各トピックを抽出
   - `Skills/`、`Vision/`、`Insights/`、`Characteristics/`にそれぞれ日付付きファイルを作成
   - 各ファイルは相互リンク

6. **おすすめアクションファイル作成**
   - 議事録から具体的なアクションアイデアを抽出
   - `Actions/`に日付付きファイルを作成
   - 優先度別（高・中・低）に分類
   - カテゴリ別（技術・ビジネス・教育・健康）に整理
   - 各アクションには理由、具体的なステップ、期待される成果を含める

## 注意事項

- 会議は建設的で建設的な分析を心がける
- 強みだけでなく改善点も議論する
- 具体的なアクションアイテムを抽出
- 各ファイルは相互にリンクして関係性を明確化
- 過去の分析との比較も実施可能
