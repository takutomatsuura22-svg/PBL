# コンテンツ発信アイデア生成

## Description
あきらパパプロジェクトの各チャンネル（YouTube、X、Facebook、RIDE ON AI、Survibe AI）ごとに「アイデア」フォルダを作成し、それぞれのコンテンツ発信に使えるmdファイルを生成します。

## Prompt
あきらパパプロジェクトの各チャンネルにコンテンツ発信アイデアのフォルダとファイルを作成してください。

## 実行ステップ

### 1. 情報収集
以下のファイルを読み取り、プロジェクトの目的と特徴を理解します：

```markdown
1. 05_Output/Areas/Content-Creation/YouTube-Channel/00-channel-strategy.md
2. 05_Output/Areas/Business/Survibe-AI/00-strategy.md
3. 04_Memory/Business/Marketing/content-marketing-strategy.md
4. 04_Memory/Business/Marketing/social-media-strategy.md
5. 04_Memory/Personal/Productivity/（関連ファイル）
```

### 2. 関連情報の収集
以下のディレクトリから最新の情報を検索：
- `04_Memory/Business/` - ビジネス戦略
- `04_Memory/Personal/` - 個人関連の知識
- `05_Output/Areas/` - 継続的な活動エリア

### 3. 各エリアにアイデアフォルダとファイルを作成

### 4. 各アイデアファイルの内容
各アイデアファイルには以下の内容を含めます：

```markdown
# [タイトル]

## 概要
[プロジェクトの目的と特徴を踏まえた概要]

## 収集した情報
### プロジェクト戦略から
- [収集した戦略情報]
- [目的・特徴]

### ユースケースから
- [関連するユースケース]
- [活用できる思考回路]

### 教材戦略から
- [教材量産戦略の考え方]
- [形式派生の活用]

## 具体的なアイデア

### アイデア1: [タイトル]
- **概要**: [アイデアの概要]
- **目的**: [達成したい目的]
- **特徴**: [このアイデアの特徴]
- **実行方法**: [具体的な実行手順]
- **期待効果**: [期待される効果]

### アイデア2: [タイトル]
...

## 実行優先順位
1. [最重要アイデア]
2. [次に実行すべきアイデア]
3. [中期的なアイデア]

## 関連ノート
- [[関連ノートへのリンク]]

#あきらパパ #コンテンツアイデア #[プロジェクトタグ]
```

### 5. 実行時の注意事項
- 既存の「アイデア」フォルダがある場合は、中身を確認してから追記するか上書きするか判断
- 各ファイルには必ず関連ノートへのリンクを含める
- 収集した情報を踏まえて、具体的で実行可能なアイデアを記載
- あきらパパプロジェクトの戦略に沿った内容にする
