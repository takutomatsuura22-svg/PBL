# MTG議事録作成

## Description
MTG議事録を作成します。議題を指定してください。

## Prompt
MTG議事録を作成してください。

引数:
- 議題: MTGのテーマ（ユーザーが指定）

要件:
- ファイル名: `01_Inbox/YYYYMMDD_mtg_[議題].md`（今日の日付を使用）
- MTGテンプレートを適用
- `#meeting` タグを追加

テンプレート:
```markdown
# YYYYMMDD_mtg_[議題]

## 参加者
- 

## 決定事項
- 

## 気づき
（重要な洞察をここに）

## 宿題
- [ ] タスク（担当者）

#meeting
```

手順:
1. 今日の日付を取得
2. 議題をファイル名に含める
3. `01_Inbox/`フォルダにファイルを作成
4. 上記テンプレートでファイルを作成
5. `#meeting`タグを追加

使用例: 「mtg ProjectX定例」
