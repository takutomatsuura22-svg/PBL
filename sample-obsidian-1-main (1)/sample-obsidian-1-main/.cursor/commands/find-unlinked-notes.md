# 孤立ノート検索

## Description
孤立ノート（バックリンク0）を検索して、リンク候補を提案します。

## Prompt
孤立ノート（バックリンク0）を検索して、リンク候補を提案してください。

実行内容:
1. `04_Memory/`内のノートをスキャン
2. バックリンク数が0のノートを抽出
3. 各孤立ノートに対してセマンティック検索で関連ノートを検索
4. リンク候補を3つずつ提案

出力形式:
```markdown
## 孤立ノート: 3件

### 1. [[ノート名A]]
リンク候補:
- [[候補1]] (関連度: 高)
  理由: 同じトピックを扱っている
- [[候補2]] (関連度: 中)
- [[候補3]] (関連度: 低)

アクション: リンクを追加しますか？
```

手順:
1. `04_Memory/`フォルダ内の全ファイルをスキャン
2. バックリンクの有無を確認
3. バックリンクが0のノートを抽出
4. 各孤立ノートに対して関連ノートをセマンティック検索
5. リンク候補を提案
6. 該当するMOC（Map of Contents）へのリンクも提案
