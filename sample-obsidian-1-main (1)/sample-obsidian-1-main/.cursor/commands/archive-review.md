# アーカイブレビュー

## Description
3ヶ月以上前のアーカイブをレビューします。

## Prompt
3ヶ月以上前のアーカイブをレビューしてください。

要件:
1. `99_Archive/` 内の全ファイルをスキャン
2. 3ヶ月以上前のファイルを抽出
3. 各ファイルの簡単な内容を表示
4. 削除候補を提案

出力形式:
```markdown
## アーカイブレビュー

### 削除候補（3ヶ月経過）
1. 2024-07/薄いメモ.md
   内容: 雑談メモ
   判断: 削除推奨

2. 2024-07/検討メモ.md
   内容: GraphQL検討
   判断: 保留（最近また話題になってる）

## アクション
- 削除: 1件
- 保留: 1件
```

手順:
1. `99_Archive/` フォルダ内をスキャン
2. ファイルパスから日付を抽出
3. 3ヶ月以上前かどうかを判定
4. ファイル内容を読み込んで簡単に要約
5. 削除候補を提案
6. ユーザーの承認を待つ

判断基準:
- 🗑️ 削除: 明らかに不要、薄い内容、重複
- ✅ 保留: まだ使える可能性がある
- 📝 Memoryへ: 見直しで価値が増した（04_Memory/へ移動）

ヒント:
- 四半期ごと（3ヶ月に1回）に実行
- 積極的に削除することが重要
- Archiveは一時保管所であり、最終保存先ではない
