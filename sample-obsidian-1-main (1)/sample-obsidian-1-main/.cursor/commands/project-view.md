# プロジェクト統合ビュー

## Description
指定プロジェクトに関連する全ノートを統合表示します。

## Prompt
指定プロジェクトに関連する全ノートを統合表示してください。

引数:
- プロジェクト名（ユーザーが指定）

実行内容:
1. `05_Output/Projects/[プロジェクト名]/`内のファイルをスキャン
2. `#project/[プロジェクト名]`タグのファイルを検索
3. カテゴリ別に整理して表示
4. 未リンクのノートを指摘

出力形式:
```markdown
# [プロジェクト名] 統合ビュー

## 📁 プロジェクトフォルダ
- [[00-project-overview.md]]
- [[01-planning/]]
- [[02-curriculum/]]
- [[03-materials/]]

## 🔗 関連ノート (タグ検索)

### MTG記録
- [[20250113-meeting-client-abc]]
- [[20250114-meeting-design-review]]

### 技術検討
- [[GraphQL導入検討]]
- [[API設計方針]]

## ⚠️ 未リンクノート
- [[孤立したメモ]]
  → 提案: [[00-project-overview.md]]とリンク
```

手順:
1. プロジェクト名を取得
2. `05_Output/Projects/@Active/[プロジェクト名]/` または `05_Output/Projects/@Planning/[プロジェクト名]/` フォルダをスキャン
3. `#project/[プロジェクト名]`タグを持つファイルを検索
4. MTG、技術検討、ドキュメントなどカテゴリ別に整理
5. 未リンクのノートを検出して提案
6. 統合ビューを表示
