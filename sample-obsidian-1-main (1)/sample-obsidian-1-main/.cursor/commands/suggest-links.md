# スマートリンク提案

## Description
現在開いているノートに関連するノートを検索して、リンク候補を提案します。

## Prompt
現在開いているノートに関連するノートを検索して、リンク候補を提案してください。

実行内容:
1. 現在のノートの内容を分析
2. セマンティック検索で関連ノートを探す
3. 関連度順に5つ提案
4. 各候補にリンク理由を付記

出力形式:
```markdown
## リンク候補: 5件

1. [[ノート名1]] (関連度: 95%)
   理由: 同じAPI設計の話題
   推奨挿入箇所: "設計原則"のセクション

2. [[ノート名2]] (関連度: 82%)
   理由: トレードオフの議論が類似

3. [[ノート名3]] (関連度: 75%)
   理由: 関連技術について言及

...
```

手順:
1. 現在開いているファイルの内容を読み込む
2. セマンティック検索で関連ノートを探す
3. 関連度の高い順に5つ選択
4. 各候補に理由と推奨挿入箇所を記述
5. 提案を表示
